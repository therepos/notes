# GitHub

Clone GitHub repo.

```git
git clone https://github.com/yourname/repo.git
```

Set Git identity in terminal.
```git
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

Stage and commit.
```git
git add .
git commit -m "Your message"
```

Push to GitHub.
```git
git push
```
